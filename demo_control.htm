Use hardware rasterization? This only applies if surface presenter is used and HW composition is currently done by the browser. If set to false, the control will render using CPU and push the result to HW using the ISurfacePresenter API.

What version of D3D to use? This version will be used to present with ISurfacePresenter if being used, and will be used for rasterization if that is set to true above.
